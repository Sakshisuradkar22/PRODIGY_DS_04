Task 04: Twitter Sentiment Analysis & Visualization
Dataset

Twitter Entity Sentiment Analysis Dataset (Kaggle)

Contains 74,681 tweets

Columns:

TweetID: Unique tweet identifier

Entity: Brand, topic, or target mentioned in the tweet

Sentiment: Sentiment label (Positive, Negative, Neutral)

Content: Text of the tweet

Objective

Analyze sentiment patterns in Twitter data to understand public opinion and attitudes toward specific brands or topics. Visualize trends in positive, negative, and neutral sentiment and identify the most discussed entities.

Tools & Libraries

Python

Pandas

NumPy

Matplotlib

Seaborn

WordCloud

Key Results

Cleaned and preprocessed 74,681 tweets

Visualized overall sentiment distribution

Identified top 10 most discussed brands/topics

Performed entity-wise sentiment analysis for top brands/topics

Generated stacked bar charts and percentage-based visualizations

Derived insights about public opinion and brand perception
